
# react-users-list

## How to run the application?

**npm**
```bash
git clone https://github.com/dcalmeida13/react-users-list.git
cd react-users-list
npm install
npm start
```
**yarn**
```bash
git clone https://github.com/dcalmeida13/react-users-list.git
cd react-users-list
yarn install
yarn start
```
